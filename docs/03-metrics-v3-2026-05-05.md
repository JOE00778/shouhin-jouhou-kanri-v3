## 一、健康度模型（ratio_months）

### 4 档阈值判定

库存月数 = `qty_on_hand / monthly_sales`

| 等级 | 标志 | 范围 | 含义 | 应对 |
|---|---|---|---|---|
| 优秀 | 🟢 | ≤ 0.7 月 | 畅销库存紧凑 | 继续补货 |
| 健康 | 🟡 | 0.7 ～ 2.0 月 | **黄金区**（Boss 拍板） | 维持稳定 |
| 注意 | 🟠 | 2.0 ～ 6.0 月 | 库存偏滞 | 促销考虑 |
| 死钱 | 🔴 | > 6.0 月 | 严重滞销 | 急处理 |

**黄金区 0.7-2.0 月**：应对订货周期 4～8 周，4 周内交货 SKU 参考，安全库存 + 销售波动容差。

### 7 条特殊规则（优先级从高到低）

| # | 条件 | 判定 | 说明 |
|---|---|---|---|
| **a** | 停售：取扱中止 OR メーカー取扱中止 | 强制 🔴 | 库存等同待清资金，等级亦强制停售 |
| **b** | C 档 + 销=0 + 1 月未动销 | 🟠 | 新增 C 档长尾，观察期 1 月 |
| **c** | C 档 + 销=0 + 连续 2 月未动销 | 🔴 | 已确认无市场，列入死钱处理候选 |
| **d** | 负库存 | 当 0 处理 | NetSuite 月度截点差，标记异常 |
| **e** | qty=0 + 销>0 | 🟢 | 断货畅销，急补货信号 |
| **f** | qty=0 + 销=0 | 🔴 金额=0 | 无库无销，清仓完成或新品尚未推 |
| **g** | C 档 + 销>0 + ratio>6 | 上限 🟠 | C 档长尾死钱仅看连续未动销时长（规则 b/c），不单看周转比 |

### 死钱金额计算

```
死钱金额 = qty_on_hand × std_cost   （仅 🔴 等级时填）
```

**含义**：资金占压。已清完商品置 0（规则 f）。

## 二、辅助指标公式

### GMROI（库存投资回报率）

```
GMROI = 月毛利 / 平均库存价值

月毛利     = sales_amount × 毛利率
平均库存价值 = (qty_on_hand + qty_on_hand_prev) / 2 × std_cost
```

**视角**：库存资金。用于评估库存投入产出比（>2.0 优秀，<1.0 预警）。

### Cross Ratio（周转与利润评分）

```
Cross Ratio = 毛利% × 月周转

月周转 = monthly_sales / avg_qty
毛利% = (sales_amount - cost_amount) / sales_amount
```

**视角**：运营。综合产品盈利能力 + 流动速度（数值越高越好）。

## 三、等级模型（4 档）

**用途**：订货决策 + 死钱信号，与健康度正交独立。频率：季度（Boss 确认按钮触发回写）。

### 判定规则

```
A 档（金牛）= 销售额前 80% × 利润率 ≥ 59%    (~200 SKU)
B 档（主力）= 销售额前 80% × 利润率 <  59%    (~1,000)
C 档（长尾）= 销售额后 20%                     (~5,000)
停售       = NetSuite 取扱区分 ∈ ('取扱中止','メーカー取扱中止')
           OR 模块③改廃確認确认
```

### 等级波动追踪

新增 `trend` 字段，每月 generate_proposal 输出：

| 符号 | 含义 | 4月首次统计 |
|---|---|---|
| ⬆️ | 升级（e.g. B→A） | 1,783 SKU |
| ➡️ | 维持 | 1,567 SKU |
| ⬇️ | 降级（e.g. A→B） | 244 SKU |

**注**：4 月升级多数因旧 ABC 标准与新算法差异，后续月份降级数会增加。

## 四、运营建议矩阵（5 档）

**仅对 B/C 档出建议**。基于毛利率 + 月周转的 3×3 矩阵：

### 阈值

| 维度 | 低 | 中 | 高 |
|---|---|---|---|
| **毛利率** | < 30% | 30～50% | > 50% |
| **月周转** | < 0.3 | 0.3～1.0 | > 1.0 |

### 5 档建议输出

| 象限 | 条件 | 符号 | 建议 | 优先级 |
|---|---|---|---|---|
| 左上 | 周转高 × 毛利低 | 🔥 | **重点提价**（快速消耗，低毛利，提价空间大） | P0 |
| 右下 | 周转低 × 毛利高 | 🔥 | **重点降价**（滞销，高毛利，通过降价刺激周转） | P0 |
| 中上 | 周转中高 × 毛利低中 | ⬆️ | **提价候选**（相对畅销，毛利提升空间） | P1 |
| 中下 | 周转中低 × 毛利中高 | ⚠️ | **降价候选**（销售不足，尝试降价提周转） | P1 |
| 左下 | 周转低 × 毛利低 | ⬇️ | **降级候选**（双低→建议下季度降档：B→C / C→停售） | P2 |
| 中中 | 周转中 × 毛利中 | ✅ | **维持**（平衡态，保持策略） | - |

### 矩阵完整图

```
                      周转 > 1.0（高）
                          │
         ⬆️提价候选          │          🔥重点提价
         (中毛×高周)       │        (低毛×高周)
                          │
毛利<30% ─────────────────┼──────────────────── 毛利>50%
         ⬇️降级候选         │         🔥重点降价
         (低毛×低周)       │        (高毛×低周)
                          │
                          │
                      周转 < 0.3（低）
```

## 五、重要术语区分 ⚠️

**常见混淆**：改廃 vs 降级

| 概念 | 改廃（改廃情報） | 降级（運営建議） |
|---|---|---|
| **定义** | 品牌方/メーカー产品迭代 | 内部数据驱动的渐变调整 |
| **信号源** | 外部（仕入先取扱中止） | 内部（销售数据 + 毛利） |
| **流程** | Page 13 改廃確認（Boss 确认） | Page 11 運営建議（自动计算） |
| **结果** | 等级强制"停售" | 等级分阶：A→B→C→停售 |
| **频率** | 不定期 + 月度汇总 | 每月 + 季度回写 |

**关键**：改廃是外部淘汰，降级是内部优化。两条线并行，不冲突。

## 六、订货公式（基于等级 + 安全系数）

```
reorder_point = monthly_sales × (lead_time_days / 30) × safety_factor

safety_factor：
  A 档 = 1.5  （高价值，必保充足，4-6 周周转风险低）
  B 档 = 1.0  （标准）
  C 档 = 0.5  （长尾，低库存策略，快速反应）
  停售 = 0    （无订货）
```

**输入**：lead_time_days 来自 supply_cycle 表（由仕入先管理リスト决定）。

## 八、真实 4 月份示例

### 示例 1：A 档 → B 档 降级

**SKU**：4987241168583（ROHTO メラノCC ビタミン C 誘導体 配合 医薬部外品 美白 美容液）

| 指标 | 值 | 说明 |

| 等级变化 | A → B | ⬇️ 降级（旧 Aランク，销量下滑） |
| 健康度 | 🟡 健康 | ratio_months = 1.2 月 |
| 运营建议 | ✅ 维持 | 毛利 42% × 周转 0.8 = 相对均衡 |

**应对**：维持 B 档，继续监控，若连续下滑考虑降 C。

### 示例 2：B 档 重点降价

**SKU**：4589546890796（&honey ヘアオイル プレミアムモイスト）

| 指标 | 值 | 说明 |
|---|---|---|
| 等级 | B 档 | 主力品 |
| 健康度 | 🟠 注意 | ratio_months = 4.5 月（偏滞） |
| 运营建议 | 🔥 重点降价 | 毛利 52% × 周转 0.2 = 高毛利低周转 |

| 建议毛利率 | 40% 目标 | 从 52% 让利 12% 刺激周转 |

**应对**：通过折扣策略释放库存，预期月周转从 0.2→0.8（数据驱动回调）。

### 示例 3：C 档 重点提价

**SKU**：4902750706200（UHA 味覚糖 コロロ ストロベリー グミ）

| 指标 | 值 | 说明 |
|---|---|---|
| 等级 | C 档 | 长尾品 |

| 运营建议 | 🔥 重点提价 | 毛利 18% × 周转 3.2 = 低毛利高周转 |

### 示例 4：B 档 死钱

**SKU**：4901008314709（CANMAKE マットシフォンリップス 03 ピンク）

| 指标 | 值 | 说明 |
|---|---|---|
| 等级 | B 档 | 曾经主力 |
| 健康度 | 🔴 死钱 | ratio_months = 37.5 月（严重积压） |

## 十一、变更说明（v2 → v3）

| 项目 | v2 | v3 | 变更 |
|---|---|---|---|
| 健康度主指标 | 进货周期 3 桶（12/6/4 次/年） | ratio_months 统一 | 简化规则，避免数据稀疏 |

| 运营建议 | 3 档草案 | 5 档（+维持 +降级候选） | 更细致，覆盖所有象限 |
| 术语规范 | 改廃≈降级 | 改廃 ≠ 降级 | **关键区分**，避免混淆 |
